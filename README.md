►►► Receita de Bolo Para buffar o Windows 10 ""by - Pereira"":

[]~(￣▽￣)~*  >>Segue as dicas!!!>>

• Win + R ---> msconfig ---> inicialização do sistema ---> Avançado ---> Ativar todos os núcleos 
	⚠ (Obs.: Isso é um overclock)
• Win + R ---> msconfig ---> Serviços ---> Ocultar todos os serviços microsoft ---> Desabilitar Todos que não sejam microsoft
	⚠ (Obs.: Se for um notebook, ou o computador possuir conponentes especiais, não se esqueça de deixar habilitado os drivers da marca do mesmo)
• Win + R ---> msconfig ---> Geral ---> Inicialização Seletiva ---> Carregar serviços do sistema + Carregar itens de inicialização
• Livre-se do One Drive, consome muito e é inconveniente
• Gerenciador de tarefas ---> Inicializar ---> Desabiliar Inicialização com o windows
• Gerenciador de tarefas ---> Serviços ---> Abrir Serviços (lá enbaixo) --->  Cliquar em cima --> Selecionar Tipo de inicialização ---> Desativado + Parar ---> Aqueles serviços que devem sofrer essas alterações são: Experiência Do Usuário Conectado a Telemetria, Serviço de Politica De Diagnóstico, Windows Search e SysMain 
	⚠ (Obs.: O Windows Search desabilitado pode interferir na transmissão em streaming na tv, e o serviço de conpartilhamento de rede do windows media player, e o SysMain ele para de tentar melhorar a máquina ao longo do tempo)
• Aplicativos em Segundo Plano ---> Desativar permissão para executar aplicativos em segundo plano, e desabilitar todos
• Configurações de notificações e ações ---> Desabilitar tudo
• Multitarefas ---> Desabilitar a Linha do Tempo
• Experiências Compartilhadas ---> Desabilitar Compartilhar entre dispositivos 
	⚠ (Obs.: Opcional)
• Para ganhar mais FPS em um momento específico apertar Control + Win + Shift + B
• No Google Chorme, Microsoft Edge ou qualquer outro navegador ---> Sistema ---> desabilitar o inicio rápido + desabilitar execução em segundo plano
• Para verificar o disco Win + R ---> Digitar mrt ---> Examinar o disco
• Para ativar o modo desenpenho do Win 10, ---> Ajustar a Aparência e o Desenpenho do Windows ---> Ajustar para obter um melhor desenpenho ---> Aplicar
• Configurações ---> Atualização e Segurança ---> Verificar se há Atualizações pendentes
• Clique em Iniciar ou na tecla Windows ---> Digite "Desfragmentas e Otimizar Unidades" ---> Selecione as suas unidades ou partições de HD ---> Clique em otimizar em cada um 
	⚠ (Obs.: Isso vai reorganizar o disco rigido de arquivos, melhorando a velocidade de acesso)
• Para Instalar mais drivers fora do Windows Instale o Driver Booster
• Para melhorar o gerenciamento das CPUs Instale o ParkControl (https://bitsum.com/parkcontrol/), lá escolha o plano de energia do Bitsum Highest Performance
• Editar plano de Energia ---> Opções de Energia (Botão estará acima) ---> Escolher Alto Desenpenho, o se tiver um melhor, como o Bitsum 
	⚠ (Obs.: Prestar bem atenção para que ele não use demais ou de menos, pois pode aumentar a temperatura base do processador)
• Para melhorar o tempo de resposta do mause Instale o ISLC (https://www.wagnardsoft.com/forums/viewtopic.php?t=1256), Em ISLC (Inteligent standby list cleaner) ---> Checar se o Current timer resolution está próximo ou abaixo de 1 (Quanto menor melhor) ---> Clique Purge Standby  list ---> Selecione "Enable custom timer resolution" ---> Em "Free memory is lower than" Digite o valor da metade da sua memoria ram, Em "Wanted timer resolution" Digite 0.5, e Em "ISLC polling Rate (ms)" Escolha 500 ---> Selecione Start ISLC minimized and auto-Start monitoring ---> Após isso pressione Start e minimize 
	⚠ (Obs.: Pode ser necessário fazer isso toda vez que for jogar) 
	⚠ (Obs.: Opcional)

►►► Receita de Bolo Para buffar o Windows 7 ""by - Pereira"":

<(￣ c￣)y▂ξ  >>Obs.: O windows 10 é baseado no windows 7, então parte das otimizações acima também servem para o windows 7>>

• Na Área de trabalho ---> Botão Esquerdo do mause ---> Personalizar ---> Selecione o tema Windows 7 Basic em Temas e Alto contraste
• Aperte a tecla windows e digite "Windows Defender" ---> Em "Ferramentas" ---> Clique em Opções ---> Desative "Verificar Meu Computador Automáticamente"
• Aperte a tecla windows e digite "services.msc", execute o arquivo ---> Em Serviços acesse a central de segurança ---> Procure "Propriedades de Central de Segurança" ---> Ao cliquar desative o "Tipo de Inicialização", em clique em ok
• Aperte a tecla windows e digite "Desfragmentador de disco" ---> Clique em "Configurar Agendamento" ---> Em "Desfragmentador de disco" procure "Alterar Agendamento" ---> Desativar "Executar seguindo um agendamento" ---> Ok
• Caso queira mais memória normal (sem ram) e seu computador possuir USB 2.0, Ative o Read Boost 
	⚠ (Obs.: Não Altera o desenpenho)

►►► Caso dê merda não se assuste, temos medidas de emergência para tal:

(* ￣︿￣)  >>Se algo de errado não estiver certo...>>

• Abra o cmd como administrador ---> execute o comando "chkdsk" (Vai verificar os arquivos do hd)
• Abra o cmd como administrador ---> execute o comando "chkdsk /f" e "chkdsk /r" (Vai repara o sistema)
• Abra o cmd como administrador ---> execute o comando "sfc /scannow" (Vai corrigir inúmeros programas e arquivos aleatórios corronpidos e tals)
• aprete a tecla windows ---> digite "mrt" ---> Selecione a opção que desejar para o exame do computador (Vai procurar virus pelo computador)
